# Task Manager Web Application

A simple web application to manage daily tasks. Users can add, edit, delete, and mark tasks as complete.

## Features

-   Add new tasks
-   Edit existing tasks
-   Delete tasks
-   Mark tasks as complete

## Technologies Used

-   HTML
-   CSS
-   JavaScript
